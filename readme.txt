== Description ==

Enable to easily embed yonderbound widget withing your site. Visitors can access the corresponding
activity or hotel from the widget.

== Installation ==

= Manual Installation =

1. Upload the entire `/yonderbound-widget` directory to the `/wp-content/plugins/` directory.
2. Activate Yonderbound Widget through the 'Plugins' menu in WordPress.
3. Enter your affiliate code in the Yonderbound setting page.
4. You can then embed the activities and hotels from the editor.
